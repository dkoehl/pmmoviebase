# Moviedatabase
* This is an TYPO3 CMS Extension for showing your movie library in a nice way.

### Requirements
* `TYPO3 CMS Versions: 6.2.x - 7.6.x`
* `EXT: Bootstrap_Package 6.2.x - 7.6.x`
* `EXT: Extbase`
* `EXT: FLUID`
